# Archived visual atmospheres (removed from UI)

These **visual** atmosphere modes were removed from the app Settings dropdown as of v0.2.9. They are documented here for reference and migration behavior.

| Legacy ID           | Label            | Notes |
|---------------------|------------------|-------|
| `wind`              | Wind & clouds    | Drifting cloud layers + wind frame. Removed from UI; saved settings migrate to **Off**. |
| `stormy-focus`      | Stormy focus     | Combined rain + wind visuals. Removed from UI; saved settings migrate to **Light rain** so users keep a storm-adjacent look. |

**Background Sound** options named similarly (e.g. wind bed, stormy focus audio) are unchanged — only the **visual** Atmosphere dropdown entries were removed.
